## e-Wallet

Simple eWallet system for Comfortable purchasing.

### Installation

**Using Composer**
---

     composer require shankarthiyagaraajan/ewallet
     
**Database Migration**
---

     php artisan migrate
     
**Seeding Database**
---

Here, seeding is required for implement the basic setups.

     php artisan db:seed --class=CreditScoreSeeder
